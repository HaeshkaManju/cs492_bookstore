# Database Migrations

This folder will contain database migrations generated by Flask-Migrate.

## Setup Instructions

1. **Initialize migrations** (first time only):
   ```bash
   flask db init
   ```

2. **Generate migration from models**:
   ```bash
   flask db migrate -m "Initial schema"
   ```

3. **Apply migrations**:
   ```bash
   flask db upgrade
   ```

4. **Rollback last migration**:
   ```bash
   flask db downgrade
   ```

## Migration Best Practices

- Always review generated migrations before applying
- Include meaningful descriptions with `-m` flag
- Test migrations on a copy of production data
- Keep migrations in version control
- Never modify applied migrations

## Current Schema

The initial migration creates these tables:

| Table | Description |
|-------|-------------|
| users | User accounts and authentication |
| customers | Extended customer profiles |
| addresses | Customer addresses |
| books | Book catalog entries |
| warehouses | Physical storage locations |
| inventory | Stock by book/warehouse/condition |
| invoices | Customer invoices |
| invoice_lines | Invoice line items |
| book_requests | Customer book requests |
| manufacturers | Book suppliers |
| purchase_orders | Orders to manufacturers |
| po_lines | PO line items |
